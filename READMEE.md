## 🚀 Proyecto Destacado

### 🔧 Predicción de Vida Útil de Motores Turbofán — NASA C-MAPSS

> Modelo de Machine Learning para predecir la **Vida Útil Restante (RUL)** de motores a partir de 21 sensores.

- 📊 **RMSE: 15,79 | R²: 0,831**
- 🤖 Algoritmo: **Random Forest Regressor** con validación cruzada e ingeniería de características
- 🧹 Preprocesamiento con **MinMaxScaler**, eliminación de variables constantes
- 📈 Series temporales: modelado de degradación secuencial
- 🌐 App web con **Streamlit** para diagnóstico, simulación y visualización de métricas
- 🗂️ Estructura modular: `notebooks/`, `src/`, `data/`, control de versiones con Git
